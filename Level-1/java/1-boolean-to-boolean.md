## AND

### definition
`and` takes two boolean values and returns a boolean value as the logical AND of the two values

```java runnable
boolean and(boolean val1,boolean val2) {
 
}
```

Solution:
```java runnable
boolean and(boolean val1,boolean val2) {
    return val1 && val2;
}
```
